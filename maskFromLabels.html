Module: Mask From Labels ()

Description:

This is a custom module that allows the user to extract portions of a dataset as described by a label field. It is intended for use in applications such as extracting the skeleton from a CT dataset, multiple organs from an MRI dataset, etc.

This module accepts a connection to the original dataset and to a Label Field, which is the type used to store segmentation results in . For each material occurring in the incoming label field a corresponding masked dataset will be created. The resultant datasets will be named based on the label names.

Select the checkbox if auto cropping is desired and press the Apply button to start the computation.

Connections:

Data [required]
The scalar field to be masked.

Label Field Data [required]
The label field to be applied as a series of masks (one per material).

Ports:

Auto crop masked data Determines whether masked data will be auto cropped. If the checkbox is not selected, the masked data will have a bounding box identical to that of the original data. If the checkbox is selected, an auto crop operation will be performed to remove excess zero values after masking. The resulting data will have the same resolution and spatial orientation as the original data but a smaller bounding box.
